# DEMO PLAN

1. Core Purpose

Demo is a free website builder that creates websites and logos based on user instructions.

2. Website Creation Flow

User starts a new website.

Demo asks the user to describe the logo.

Demo generates the logo exactly as described.

Demo builds the website using the logo’s style (colors, fonts, layout).

User customizes pages and content.

Demo publishes the website.
